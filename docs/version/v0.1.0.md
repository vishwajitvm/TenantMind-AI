# Release Notes v0.1.0

This release marks the initial architecture baseline for TenantMind AI.

## 1. Components Included
* **Backend**: FastAPI structure incorporating tenancy contexts and model integrations.
* **Frontend**: Next.js client with Zustand store configurations, Tailwind, Lucide Icons, and `@xyflow/react`.
* **Queues**: Celery and Redis configurations for handling async PDF processing, invoicing, and reminders.
* **Authentication**: Keycloak realm security with OIDC roles checks.
* **Vector Store**: Qdrant schema definitions for lease document RAG search.
