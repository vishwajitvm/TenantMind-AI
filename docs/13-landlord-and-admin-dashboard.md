# 13. Landlord & Admin Dashboard

The Landlord Dashboard provides property managers with high-level control over properties, financial flows, maintenance tickets, and AI activity.

## 1. Property & Financial Analytics
* **Yield Performance**: Renders charts illustrating monthly rent collection, outstanding balances, and maintenance costs.
* **Property Register**: Landlords can add new buildings, manage unit inventories, and assign new tenants to leases.

---

## 2. Maintenance Triage Board (Kanban)
* **Status Columns**: Drag-and-drop workflow tracking tickets from `OPEN` to `ASSIGNED`, `RESOLVED`, and `CLOSED`.
* **Vendor Directory**: Quick-assign dropdowns matching categories (e.g. assigning a plumbing issue to a local vendor specialized in plumbing).

---

## 3. MCP Tool Approval Console
A unique security interface that shows AI-initiated tool requests.
* **Risk Categorization**: Flags tasks by risk level (Medium, High, Critical).
* **Control Actions**: Landlords can click **Approve** (executes the action) or **Deny** (cancels and logs the denial) for any pending high-risk tool call.
